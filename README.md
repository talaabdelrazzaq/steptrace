# StepTrace
I built this because I kept blanking on algorithms I thought I understood. Most times reading a solution is not the same as watching it work visually.

StepTrace walks you through common coding interview algorithms one step at a time — the code highlights on the left, the data structure updates live on the right. Hit Play and watch.

## Algorithms

- **Two Sum** — hash map approach
- **Binary Search** — divide and conquer
- **Valid Parentheses** — stack

## Stack

Next.js, React, Tailwind CSS, Framer Motion

## Run it locally
```bash
npm install
npm run dev
```

## What's next
